# Productivity Dashboard

This is a static productivity dashboard site suitable for GitHub Pages hosting.

## Deployment

1. Push this repository to GitHub.
2. Enable GitHub Pages in repository settings.
   - Source: `main` branch
   - Folder: `/ (root)`
3. Visit the published URL shown in GitHub Pages.

## Notes

- The site entry point is `index.html`.
- Assets are served from the root and `fav/` folder.
- Weather data requires a valid `apiKey` in `script.js`.

## Files

- `index.html`
- `style.css`
- `script.js`
- `fav/site.webmanifest`
- `fav/favicon.ico`
- Font files: `AeonikTRIAL-*.otf`
